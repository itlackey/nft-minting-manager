#Getting Started

Follow these steps to debug the contract tests in VSCode

1. Open this folder in VS Code
1. Run `npm install` in terminal
1. Run `truffle migrate`
1. Press `F5`

Or from the CLI

1. Open this folder in VS Code
1. Run `npm install` in terminal
1. Run `truffle migrate`
1. Run `truffle develop`
1. Run `test` in the truffle console
